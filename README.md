# 🕹️ Catch The Kenny

A fast-paced, interactive mini-game developed with **Swift** and **UIKit**. This project demonstrates the fundamental concepts of iOS development, including timer management, user interaction, and dynamic UI updates.

## 🛠 Features
- **Real-time Interaction:** Capturing user taps on moving objects.
- **Timer Logic:** Implementing `Timer` for both game duration and object movement speed.
- **Score Tracking:** Local score management and high-score persistence using `UserDefaults`.
- **Alert Management:** Dynamic game-over alerts with replay functionality.

## 💻 Tech Stack
- **Language:** Swift
- **Framework:** UIKit
- **Architecture:** MVC (Model-View-Controller)

## 🚀 Learning Outcomes
Through this project, I practiced:
- Handling **UIImageView** and **UITapGestureRecognizer**.
- Managing asynchronous tasks with **NSTimer**.
- Designing responsive layouts with **Auto Layout**.
